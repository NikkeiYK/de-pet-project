Производственные метрики:
OEE (Overall Equipment Effectiveness) по экструдерам/реакторам
Yield (выход продукта) в зависимости от параметров процесса
Прогноз образования кокса в реакторах (на основе телеметрии)

Качество продукции:
Соответствие партий спецификациям (процент брака)
Корреляция параметров процесса с лабораторными тестами

Экономика:
Стоимость простоя оборудования (потери в рублях)
Эффективность использования катализаторов (выход продукта на единицу катализатора)

Историчность:
Хранить историю изменений параметров реакторов (после модернизации) — SCD Type 2
Хранить историю изменения спецификаций продукции — SCD Type 2
